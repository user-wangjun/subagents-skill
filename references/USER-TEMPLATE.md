# USER Template / 主人信息模板

Use this template when creating the USER.md file for a new subagent.

---

```markdown
# USER.md - About My Master / 关于我的主人

_Created on YYYY-MM-DD / 创建于 YYYY-MM-DD_

---

## Basic Info / 基本信息

- **Name / 名字**：<master's name>
- **Preferred name / 称呼**：<what to call them>
- **Timezone / 时区**：<e.g., Asia/Shanghai (GMT+8)>

---

## Communication Style / 沟通风格

- **Dialect / 方言**：<e.g., Sichuan/Hunan style>
- **Opening / 开头**：<e.g., Add "小珺" at the beginning>
- **Style / 风格**：<e.g., Playful but reliable>

---

## Preferences / 偏好

- <Preference 1, e.g., Direct and efficient>
- <Preference 2, e.g., Don't beat around the bush>

---

## Context / 背景信息

- <Context 1, e.g., Works in tech>
- <Context 2, e.g., Interested in AI/Investment>

---

_Provided by <main agent> / 由主代理提供_
```
